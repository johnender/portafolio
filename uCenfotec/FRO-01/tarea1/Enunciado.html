Tarea 1: Ejercicios JS

Objetivos

El objetivo de esta tarea es que el o la estudiante practique Javascript, tanto el lenguaje en general como las diferentes formas de manipular el DOM.

El o la estudiante deberá crear resolver los siguiente ejercicios:

  1. Modifique el estilo, como usted desee, del párraforo del archivo ejercicio 1. Respuesta
  2. Obtener el valor de los campos de nombre y apellidos e imprimirlos en la consola. Respuesta
  3. Escriba un código en Javascript que cambie el color de fondo de un párrafo. Respuesta
  4. Escriba un programa JavaScript para obtener el ancho y el alto de la ventana (cada vez que se cambia el tamaño de la ventana). Respuesta
  5. Escriba un programa JavaScript para mostrar una imagen aleatoria (haciendo clic en un botón) de la siguiente lista. Información de la imagen de muestra:
    • "http://farm4.staticflickr.com/3691/11268502654_f28f05966c_m.jpg", ancho: "240", alto: "160"
    • "http://farm1.staticflickr.com/33/45336904_1aef569b30_n.jpg", ancho: "320", alto: "195"
    • "http://farm6.staticflickr.com/5211/5384592886_80a512e2c9.jpg", ancho: "500", alto: "343"
    6. Respuesta
  6. Escribe un programa JavaScript para calcular el volumen de una esfera. Respuesta
  7. Escriba un programa de JavaScript para eliminar el elemento seleccionado de una lista desplegable. Respuesta
  8. Escriba una función de JavaScript que cree una tabla, acepte filas, números de columna del usuario e ingrese el número de fila-columna como contenido (por ejemplo, Fila-0 Columna-0) de una celda. Respuesta
  9. Escriba una función de JavaScript que acepte filas, columnas (para identificar una celda en particular) y un string para actualizar el contenido de esa celda. Respuesta
